# 任务 4.4.1 实现总结：前端审批节点编辑器 - 驳回策略选择

### 📝 使用说明

#### 在流程设计器中使用

1. **创建审批节点**
   - 在流程设计器中添加审批节点
   - 选择节点类型为"人工审批"或"自动节点"

2. **配置驳回策略**
   - 在右侧节点编辑器中找到"驳回策略"分区
   - 从下拉菜单中选择：
     - "驳回到发起人"：驳回后流程结束，发起人可重新提交
     - "驳回到上一个审批节点"：驳回后返回上一个审批人重新审批

3. **保存配置**
   - 配置自动保存到流程草稿
   - 发布流程时生效

#### 驳回策略的实际效果

**TO_START 策略**：
```
发起人 → 部门经理(TO_START) → 驳回 → 流程结束，发起人重新提交
```

**TO_PREVIOUS 策略**：
```
发起人 → 部门经理 → 总经理(TO_PREVIOUS) → 驳回 → 返回部门经理重新审批
```
