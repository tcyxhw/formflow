## 2.2 系统相关技术介绍

### 2.2.1 FastAPI

FastAPI是一个现代化的Python Web框架，由Sebastian Ramirez于2018年创建[1]。FastAPI基于Pydantic进行数据验证，基于Starlette处理ASGI应用，其核心理念是通过Python类型注解实现自动化的数据验证和API文档生成。FastAPI的异步编程模型能够高效处理高并发场景下的IO密集型请求，配合Pydantic的数据验证机制，能够确保API输入输出数据的类型安全。

### 2.2.2 Python

Python是一种解释型、面向对象的高级编程语言，由Guido van Rossum于1991年首次发布[2]。Python的设计哲学强调代码的可读性和语法的简洁性。在Web开发领域，Python生态中有Django、Flask和FastAPI三大主流框架。Python 3.12版本在性能方面有显著提升，其丰富的第三方库生态为系统提供了可靠的技术支撑。

### 2.2.3 PostgreSQL

PostgreSQL是一个功能强大的开源对象关系型数据库管理系统，由加州大学伯克利分校于1986年首次发布[3]。PostgreSQL遵循ANSI SQL标准，其MVCC（多版本并发控制）机制能够有效支持高并发读写场景。PostgreSQL的JSONB类型能够高效存储表单的动态字段结构，同时支持索引查询。

### 2.2.4 SQLAlchemy

SQLAlchemy是Python生态中最成熟的ORM框架，于2005年首次发布[4]。SQLAlchemy的设计理念是提供一套将Python对象映射到数据库表的工具，同时保留SQL的完整表达能力。SQLAlchemy 2.0版本对API进行了重大重构，提供了更加直观的CRUD操作接口和更强的类型安全保证。

### 2.2.5 Redis

Redis是一个开源的内存数据结构存储系统，由Salvatore Sanfilippo于2009年首次发布[5]。Redis支持字符串、哈希、列表、集合、有序集合等多种数据结构，同时支持Lua脚本、事务和持久化等高级特性。由于数据存储在内存中，Redis的读写性能极高。在Web应用架构中，Redis主要承担缓存和会话存储两类角色。

### 2.2.6 Vue 3

Vue.js是一套用于构建用户界面的渐进式框架，由尤雨溪于2014年首次发布[6]。Vue 3采用Proxy代理对象重写了响应式系统，性能比Vue 2的Object.defineProperty方案提升近一倍。Vue 3的Composition API提供了更加灵活的组件逻辑组织方式。Vue的组件化结构使得前端开发更加高效、可维护。

### 2.2.7 Naive UI

Naive UI是一个基于Vue 3的企业级UI组件库，由TUIMOO团队开发和维护[7]。Naive UI提供了丰富的PC端��件，包括表单组件、表格组件、布局组件、反馈组件等。Naive UI的Form组件提供了完整的表单验证能力，DataTable组件在大数据量展示方面性能良好。

### 2.2.8 Pinia

Pinia是Vue官方推荐的状态管理库，由Eduardo San Martin Morote于2019年创建[8]。Pinia采用Store概念组织状态，每个Store是一个独立的响应式状态容器。Pinia在保持了Vuex核心功能的同时，提供了更加简洁的API设计和更好的TypeScript支持。

### 2.2.9 ECharts

ECharts是百度开源的可视化图表库，于2013年首次发布[9]。ECharts提供了丰富的图表类型，包括折线图、柱状图、饼图、散点图、K线图、地图等。ECharts采用Canvas和SVG双引擎设计，能够在保持高性能的同时，提供灵活的交互能力。

### 2.2.10 Vue Flow

Vue Flow是Vue 3的流程图可视化库，基于@xyflow/react开发[10]。Vue Flow提供了节点、边、自定义组件等核心概念，支持拖拽布局、交互连线、局部缩放和平移等功能。Vue Flow能够直观地展示审批流程的节点和分支结构。

### 2.2.11 Vite

Vite是Vue.js官方推荐的构建工具，由尤雨溪于2020年创建[11]。Vite利用浏览器原生ES模块实现开发时的极速热更新，利用Rollup实现生产环境的优化构建。Vite的启动速度是毫秒级的，不需要像传统打包工具那样预先扫描整个项目。

### 2.2.12 其他关键技术

除上述核心技术外，本系统还涉及以下关键技术：MinIO是一个开源的对象存储系统，兼容Amazon S3 API，本系统采用MinIO存储用户上传的表单附件。APScheduler是Python的定时任务调度库，本系统采用APScheduler实现草稿清理和过期数据统计等后台任务。JWT是开放标准定义的令牌格式，本系统采用JWT实现无状态的Token认证。智谱AI（GLM）是国产的大语言模型，本系统集成智谱AI实现智能表单生成功能。
