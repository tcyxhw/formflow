6.1 提交数据格式（data_jsonb）
TypeScript

interface SubmissionData {
  // 文本类字段
  name: string;
  email: string;
  
  // 数字类字段
  amount: number;
  
  // 选择类字段（单选存value）
  department: string;
  
  // 多选类字段（存value数组）
  tags: string[];
  
  // 日期范围字段
  date_range: {
    start: string;  // YYYY-MM-DD
    end: string;
  };
  
  // 时间范围字段
  time_range: {
    start: string;  // HH:mm:ss
    end: string;
  };
  
  // 附件字段（存附件ID数组）
  attachments: number[];
  
  // 布尔值字段
  agree: boolean;
  
  // 计算字段（自动计算的结果也会保存）
  total_amount: number;
}
前端填写格式 vs 后端存储格式:

字段类型	Naive UI 格式	后端存储格式
日期范围	["2024-01-01", "2024-01-07"]	{"start": "2024-01-01", "end": "2024-01-07"}
时间范围	["09:00:00", "17:00:00"]	{"start": "09:00:00", "end": "17:00:00"}
附件	[{id: 123, name: "file.pdf", ...}]	[123, 456]
需要转换：提交前将 Naive UI 格式转为后端格式

七、页面功能需求
7.1 表单填写页
路由: /forms/{id}/fill

核心功能:

获取表单配置并渲染
获取草稿数据（如果有）
每30秒自动保存草稿
文件上传
表单验证
提交数据
关键逻辑:

TypeScript

// 1. 页面初始化
onMounted(async () => {
  // 获取表单
  const formRes = await api.getFormDetail(formId);
  formConfig.value = formRes.data;
  
  // 获取草稿
  const draftRes = await api.getDraft(formId);
  if (draftRes.data) {
    formData.value = draftRes.data.draft_data;
  }
  
  // 启动自动保存
  autoSaveTimer = setInterval(saveDraft, 30000);
  startTime = Date.now();
});

// 2. 保存草稿
const saveDraft = async () => {
  await api.saveDraft({
    form_id: formId,
    data: formData.value
  });
};

// 3. 提交表单
const submit = async () => {
  clearInterval(autoSaveTimer);
  
  // 转换数据格式（日期范围等）
  const submitData = transformToBackend(formData.value);
  
  await api.createSubmission({
    form_id: formId,
    data: submitData,
    duration: Math.floor((Date.now() - startTime) / 1000),
    source: 'pc'
  });
};
数据格式转换:

TypeScript

// Naive UI → 后端
const transformToBackend = (data) => {
  const result = {...data};
  
  // 日期范围：[start, end] → {start, end}
  if (Array.isArray(data.date_range)) {
    result.date_range = {
      start: data.date_range[0],
      end: data.date_range[1]
    };
  }
  
  // 附件：[{id, name, ...}] → [id1, id2]
  if (Array.isArray(data.attachments) && data.attachments[0]?.id) {
    result.attachments = data.attachments.map(f => f.id);
  }
  
  return result;
};

字段值格式化:

TypeScript

const formatValue = (fieldId, value, snapshot) => {
  const options = snapshot.field_options[fieldId];
  
  // 选项类字段
  if (options) {
    if (Array.isArray(value)) {
      return value.map(v => 
        options.find(o => o.value === v)?.label || v
      ).join(', ');
    } else {
      return options.find(o => o.value === value)?.label || value;
    }
  }
  
  // 日期范围
  if (value?.start && value?.end) {
    return `${value.start} ~ ${value.end}`;
  }
  
  // 附件
  if (Array.isArray(value) && typeof value[0] === 'number') {
    return `${value.length} 个文件`;
  }
  
  return value;
};

const handleFinish = ({ file, event }) => {
  const response = JSON.parse(event.target.response);
  
  if (response.code === 200) {
    file.id = response.data.id;
    file.url = response.data.download_url;
    file.status = 'finished';
  } else {
    file.status = 'error';
    message.error(response.message);

};
九、异常处理
常见错误码
错误码	说明	处理方式
4001	验证错误	显示错误信息，标红对应字段
4002	业务错误	弹窗提示，如"表单未发布，无法提交"
4031	权限不足	提示无权限
4041	资源不存在	提示资源不存在，返回列表页
5001	服务错误	提示"系统错误，请稍后重试"
常见业务错误
错误信息	场景
"表单未发布，无法提交"	提交草稿状态的表单
"已超过提交截止时间"	超过 submit_deadline
"该表单不允许提交后编辑"	allow_edit=false
"已超过最大编辑次数"	超过 max_edit_count
"只能编辑自己的提交"	编辑他人提交
"文件大小超过限制（最大10MB）"	上传文件过大
"不支持的文件类型"	上传不允许的文件类型
