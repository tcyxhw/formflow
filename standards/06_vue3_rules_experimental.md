# Vue3 Experimental Rules

## 候选规则 1：页面状态最小闭包化
适用：
- 表单页
- 列表筛选页
- 复杂交互页

要求：
- 页面局部状态优先放在页面或 composable 内部
- 非必要不提升到全局 store

## 候选规则 2：接口层响应归一化
适用：
- 多页面复用接口
- 复杂响应结构映射场景

要求：
- 优先在 API 层做格式归一
- 组件层尽量消费稳定数据结构

## 候选规则 3：副作用集中管理
适用：
- onMounted 初始化请求
- watch 联动请求
- 表单自动保存

要求：
- 副作用逻辑集中在既有约定位置
- 避免模板、样式、渲染分支中夹带业务副作用
